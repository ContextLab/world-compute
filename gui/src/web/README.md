# World Compute Web Dashboard

This directory will contain the World Compute web dashboard — a React single-page application (SPA) served from a static CDN.

## Status

Not yet implemented. This is a placeholder.

## Planned Stack

- React (with TypeScript)
- Vite for bundling
- Deployed as static assets to a CDN (e.g., Cloudflare Pages or similar)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The output will be a `dist/` directory of static files suitable for CDN deployment.
